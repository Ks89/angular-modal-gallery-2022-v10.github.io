Keyboard configuration

If you want to customize keyboard download shortcuts, you can pass a custom configuration to libConfig.


KeyboardServiceConfig

Interface to change keyboard download shortcuts.


shortcuts
Type: string[] | string

This is a direct input into KeyboardService (mousetrap.bind(HERE, ...)) to manage keyboard's shortcuts. Please refer to the official documentation of mousetrap. By default this value is ['ctrl+s','meta+s'].
disableSsrWorkaround
Type: boolean

If you are using SystemJS, instead of angular-cli or webpack, you must set this to true. By default this value is false.

This is an advanced configuration property and it should be used very carefully only by SystemJS users.

I cannot support SystemJS forever. Please, consider to upgrade your project to angular-cli as soon as possible.
