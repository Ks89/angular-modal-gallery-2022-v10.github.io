Image

The main object that represents an Image with both modal, thumbnail, description and so on. The first thing to do is creating an array of Image instances (using the new keyword). Every element represents a single image.


id
Type: number

Mandatory UNIQUE id to identify the image.
modal
Type: ModalImage

Mandatory interface to configure the modal image.
plain
Type: PlainImageoptional

Optional interface to configure the thumbnail (or 'plain image').
